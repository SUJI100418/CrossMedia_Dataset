# CrossMedia_Dataset : 경의로운 소문(Naver Talk 채팅, 다음 웹툰 댓글)
<br>
## 1. 웹툰 데이터

‘흥미로운(Interesting)’ 이야기와 ‘흥미 없는 또는 지루한(Uninteresting or Boring)’ 이야기의 기준이 될 수 있는 척도를 설정하기 이전에 이야기의 흥미여부를 객관적으로 판단하기 위한 데이터로서 웹툰의 댓글(Comment)을 선택한다.

웹 크롤러(Web crawler)를 이용하여 2021.02.20 일 기준 시즌1 의 60 화와 시즌2 의 44 화, 그리고 에필로그 에피소드를 포함하여 총 105 개의 에피소드를 크롤링하였으며, 그 결과 총 34,594 개의 댓글 데이터를 수집하였다.

- 에피소드 댓글의 특징 : 웹툰 자체의 내용보다는 사회적 이슈, 휴재, 연재 등의 소식에 관해 댓글이 다수임

## 2. 드라마 데이터

네이버 실시간 Talk 는 특정 프로그램에 대한 시청자 채팅 기능을 제공하기 때문에 TV 프로그램에 대한 시청자 반응을 실시간으로 수집할 수 있다. 

총 433,066 문장의 ‘경이로운 소문(드라마)’에 대한 실시간 Talk 데이터를 수집하였다. 그 중, 정규 편성된 생방송 시간에 해당하는 데이터는 300,938 문장(69.49%)이며, 나머지 132,128 문장은 재방송 시간에 생성되었다. 재방송 시간에 생성된 데이터는 스포일러를 포함하거나 광고, 도배성 채팅이 강하기 때문에 분석 데이터에서 제외하였다. 

### 경이로운 소문(드라마) 실시간 Talk 데이터 예시
<img src="https://user-images.githubusercontent.com/46912893/149653605-8ee9007e-baf4-41b5-b131-254b4574e73e.JPG" width="60%" height="60%">

가장 많이 실시간 Talk 데이터가 생성된 에피소드는 마지막화인 16 화이며, 가장 적은 에피소드는 1 화이다.
전체적으로 Talk 데이터 생성량이 증가하는 추세이나, 10 화 이후 Talk 데이터 생성량이 급격히 줄어들었다가 12 화 이후 다시 증가하였다.
